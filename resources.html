Learning Resources

Curated books, courses, tools, and communities to accelerate your AI learning journey.

We've handpicked the best free and paid resources to help you learn AI effectively. Whether you're just starting out or advancing to production systems, you'll find something valuable here.

Books

Beginner

Artificial Intelligence: A Modern Approach

The definitive textbook by Stuart Russell and Peter Norvig. Covers AI fundamentals, search, logic, and machine learning comprehensively.

Learn more
Intermediate

Hands-On Machine Learning

Aurélien Géron's practical guide to ML with Scikit-Learn, Keras, and TensorFlow. Perfect for hands-on learners who want to build projects.

Learn more
Advanced

Deep Learning

By Ian Goodfellow, Yoshua Bengio, and Aaron Courville. The comprehensive reference for neural networks and deep learning theory.

Learn more
LLMs

Build a Large Language Model (From Scratch)

Sebastian Raschka's guide to understanding and building LLMs from the ground up. Essential for modern AI practitioners.

Learn more

Online Courses

Free

fast.ai Practical Deep Learning

Top-down approach to deep learning. Build models from lesson one, then understand the theory behind them.

Learn more
Free

Andrew Ng's Machine Learning Course

The course that launched millions of ML careers. Covers fundamentals with clear explanations and practical assignments.

Learn more
Free

Stanford CS229: Machine Learning

Rigorous academic course covering ML theory, algorithms, and mathematical foundations from Stanford University.

DeepLearning.AI Short Courses

Focused courses on prompt engineering, LangChain, RAG, and AI agents. Great for staying current with latest techniques.

Tools & Libraries

Python

Scikit-learn

The go-to library for classical machine learning. Simple API for classification, regression, clustering, and preprocessing.

Deep Learning

PyTorch & TensorFlow

Leading deep learning frameworks. PyTorch for research flexibility, TensorFlow for production deployment at scale.

LLMs

LangChain & LlamaIndex

Frameworks for building LLM applications with RAG, agents, and tool use. Essential for modern AI development.

Models

Hugging Face

Access thousands of pre-trained models, datasets, and spaces. The GitHub of machine learning.

Keep Learning
