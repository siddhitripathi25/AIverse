    DeepLearning.AI Short Courses

    Course Various Instructors (DeepLearning.AI) Free

    DeepLearning.AI Short Courses is a curated collection of bite-sized, practical courses developed in partnership with industry leaders like OpenAI, LangChain, LlamaIndex, and Microsoft. Taught by Andrew Ng and other experts, these courses enable developers to master cutting-edge generative AI capabilities in under two hours.

    About this Course

    This series consists of short, highly modular courses designed to teach new tools and techniques. Each course is accompanied by an online Jupyter notebook environment, allowing students to experiment with API calls, prompt adjustments, and agent flows directly as they watch the lectures.

    Key Topics Covered

    • ChatGPT Prompt Engineering for Developers (OpenAI best practices)
    • LangChain for LLM Application Development & Agent creation
    • Retrieval Augmented Generation (RAG) & Semantic Search Databases
    • Building Multi-Agent Systems & Conversational Interfaces
    • Finishing and Deploying LLM Apps with Guardrails
    • Fine-Tuning Open Source LLMs (Meta Llama, Mistral)

    Why this course is useful

    The pace of Generative AI is extremely fast. These courses cut through the noise, offering concrete, developer-focused code walkthroughs that can be completed in an afternoon. It is the fastest way to learn how to implement features like RAG, function calling, or vector search.

    Who should take it

    Software engineers, developers, and tech builders who want to learn how to apply LLMs and build products. Basic Python programming and API usage skills are recommended.
